# Chronicle

Chronicle is a web-based calendar app focused on calendar syncing, managing and smart scheduling.

## Userguide
See `docs/guides/user.md`for more informations.


## Quickstart of the containers
**Start: **

```bash
podman-compose up
```

**Stop: **
```bash
podman-compose down
```
