# Docs/User

Chronicle is a web-based calendar app focused on calendar syncing, managing and smart scheduling.

# Chronicle User Guide (Quick Start)
This User Guide will provide the informations on how to:
- create,
- manage and
- schedule events

inside chronicle.


## Description - Core Features
Chronicle is based on a few core principles:
- **Calendar Profiles**: Containers for events (e.g., Work, Personal, Team).
- **Calendar Sources**: External calendar sources which can be used to sync tasks and apppointments.
- **Events**: Appointments or tasks.

- **Scheduler? wie heißt das tool/im dashboard**: Tool, that helps you to find suitable time slots for your preferences based on calendar availability.

## Usage
### Sign in/up
To use chronicle you need an user account (email and password). If you already have one just sign in.

## What you can do
**Calendar Profil**: Create one or multiple of these to keep track of things.
- Creating multiple of these profiles for various applications help to maintain an overview and clearly separate the application cases.
- e.g. Work, University, Personal, Team, ...


**Sync a calendar**: Connect external calendars (via iCal or CalDAV) with the **Calendar Sources** feature to sync them with your current chronicle calendar profil.


## Calendar Management
### Smart Schedulin Feature
Chronicle Events can be smart scheduled. So you can input something like:
- Sport, 3 times a week for one hour.

With this information chronicle finds free slots, creates these appointments and syncs these with your connected calendars.

In addition to this example, it is also possible to create tasks with deadlines what for then regular (study) sessions will be created with pre specified durations.

### Sync Between Chronicle and external Sources
The sync from chronicle to the extarnal calendar is only possible if the external calender supports CalDAV.
If this is not the case you can use a chronicle export link (iCal/CalDAV) to sync your appointsments back on your devices.
