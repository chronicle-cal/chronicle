# Chronicle Documentation

## Structure

- **Overview**: General overview of the project, its goals and core features.
- **User Documentation**: Guide for users on how to use the app.
- **Developer Documentation**: Guide for developers on how to set up the project, contribute and understand the architecture.
- **Admin Documentation**: Guide for admins on how to manage the app and its users.

**Architecture**
`docs/architecture/decisions`: Decisions of the arcitecture in order to understand them.
